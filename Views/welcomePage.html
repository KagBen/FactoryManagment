Welcome to Factory Management

Welcome to the Factory Management web application, a comprehensive solution for efficiently managing your factory operations. Here's what sets our system apart:

  • Secure Login: Users log in securely with JWT-based authentication.
  • Detailed Employee Information: View employees' detailed information, including their shifts.
  • Employee and Department Management: Create, edit, or delete employees and departments effortlessly.
  • Efficient Allocation: Easily allocate employees to their respective departments.
  • Shift Management: Manage shifts and assign employees efficiently for optimized workforce management.
  • Action Limits: Set daily action limits for users to maintain system integrity.
  • Automatic Logout: Users are logged out automatically if they exceed their daily action limits, ensuring fair usage.
  • User Profiles: Access user profiles with action limit details for effective monitoring and user management.
  • Role-Based Access: Admins have the privilege to fetch all users, ensuring controlled access to user data.

Our user-friendly interface, robust security features, and RESTful architecture guarantee efficient factory management. With Factory Management, you're in control of your factory operations. Get started today!

Technology Stack:

  • Server: Node.js and Express.js for building a robust backend.
  • Database: MongoDB for efficient data storage and retrieval.
  • Frontend: HTML, CSS, and JavaScript for a seamless user interface.
  • JWT Authentication: Secure login and user management using JSON Web Tokens , roles managment with jwt.
  • Session Management: Implementing server-side sessions for user interaction.
  • Cookie Management: Handling cookies for a personalized user experience.
  • RESTful API: Designing a RESTful API for seamless communication.
